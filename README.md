اسم المشروع: Advanced Text Processing with Regex
​المفهوم: البحث عن أنماط محددة داخل النصوص ومعالجتها بدقة عالية.
​هيكل المشروع:
​استيراد مكتبة re واستخدام الدالة search للبحث عن أنماط معينة.
​استخدام الدالة findall لاستخراج كافة النصوص التي تطابق نمطاً معيناً (مثل الأرقام أو الكلمات).
​التعامل مع النصوص العربية وتصفيتها باستخدام الأنماط النمطية (مثل استخراج الحروف من [ا-ي]).
